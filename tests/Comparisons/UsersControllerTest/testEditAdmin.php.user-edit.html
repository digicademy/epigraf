Username

The username should contain the real name of the person. Only alphanumeric characters are allowed. Maximum length is 50 characters.

Role
Name

First and last name.

Acronym

First letters of the name.

Email address

Please provide an email address to contact the user.

Contact

Please provide additional information about how to contact the user, for example the affiliation.

Password

Please enter a strong password and remember it well. The password can't be restored. If you loose it, you have to enter a new password. It must be at least 8 characters long and include an uppercase letter, a lowercase letter, a number, and a special character.

Default Database

Select the database that should open after you login.

Default Article Pipeline

Only Epigraf-Desktop users: Select the pipeline that exports single articles.

Default Book Pipeline

Only Epigraf-Desktop users: Select the pipeline that exports a complete book.

Language

The language in which Epigraf talks to you.

Theme

The user interface appearance.

IRI fragment

Usually the IRI equals the username. The IRI is used to match users in the main database to users in the project databases.
